# Seguridad de recomendaciones

- Dolor, mareo o dolor torácico pausan cualquier sesión automática.
- Mareo o dolor torácico muestran orientación para atención médica o urgente según intensidad.
- Fatiga, sueño deficiente, estrés alto, molestia muscular alta o disposición baja reducen el volumen 25 % y limitan RPE a 6.
- La progresión semanal modifica una sola variable y no supera 10 %.
- Embarazo/lactancia, trastorno alimentario y condición renal/metabólica desactivan cálculos nutricionales.
- Las alergias son exclusiones estrictas del catálogo de recetas.
- OpenAI sólo explica una recomendación previamente calculada; no tiene autoridad para modificarla.
